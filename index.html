Instrucciones de Uso

Este controlador web permite utilizar gestos faciales para generar señales MIDI y controlar instrumentos virtuales, así como generar sonido directamente en el navegador.

Modos de Uso:

Modo Theremin y Percusión

En estos modos, puedes escuchar el resultado sonoro directamente en el navegador utilizando la lógica de los gestos.

Theremin:

Al activar el modo Theremin, puedes elegir entre dos opciones:

  • Synth: Utiliza un oscilador. Puedes seleccionar la forma de onda (sinusoidal, cuadrada, sierra, triangular) y controlar la frecuencia abriendo y cerrando la boca, y cambiar la dinámica con la apertura de ojos.
  • Notas: Activa automáticamente el botón Notas del gesto Boca. Puedes seleccionar un instrumento MIDI para escuchar. Este modo utiliza la salida MIDI de notas generada por la boca y reproduce los sonidos directamente en el navegador.
Percusión:

Al activar Percusión, todos los botones de Disparador se activan automáticamente. Los movimientos rápidos de los gestos disparan la percusión. Usa Sensibilidad para decidir cuánto movimiento hace falta.

Salida MIDI

Si no están activados los modos Theremin o Percusión, el sistema funciona como un controlador MIDI. Puedes utilizar los gestos para enviar señales MIDI de Control, Notas o Disparadores según las configuraciones establecidas.

Grid de Performance

El modo Grid de Performance separa el apuntado de nota del disparo de nota. Mueve la nariz para elegir una celda del grid (pitch), luego abre la boca o cierra el guiño (según Fuente de Gate) para iniciar la nota y suelta para detenerla.

Ajusta Escala, Fundamental y el rango Min/Max calibrado de cada gesto para personalizar la respuesta del sistema.

Funciones de los Gestos:

  • Notas: Toca notas discretas mapeando el movimiento calibrado del gesto al rango de notas, fundamental y escala seleccionados.
  • Control: Envía valores continuos MIDI CC basados en el gesto.
  • Disparador: Activa eventos cuando se detecta un movimiento rápido del gesto.

Ajuste de Valores y Sensibilidad:

Modifica los valores Min y Max para calibrar el rango de detección de cada gesto según tus características faciales.

Lógica de Activación de Notas:

En modo Notas, el valor crudo actual del gesto se interpreta mediante calibración y se mapea al rango musical seleccionado.

  • Un valor crudo cerca del Min calibrado mapea a la primera nota permitida.
  • Un valor crudo cerca del Max calibrado mapea a la última nota permitida.
  • Cambiar fundamental o escala cambia las notas permitidas sin cambiar la calibración.
